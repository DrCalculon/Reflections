When would you want to use a remote repository rather than keeping all your work
local?

    It can help synchronise work on different computers
    It also works as a more permanent backup of the code

Why might you want to always pull changes manually rather than having Git
automatically stay up-to-date with your remote repository?

    If you are adding code to your local repository, then it might keep overwriting
    the changes that you make.  It also allows others to change and push without
    stuffing you up

Describe the differences between forks, clones, and branches.  When would you
use one instead of another?

    Forking is for making a copy of someone else's repository
    Cloning is for making a copy of your own
    Branching is more making a different version of your own codebase
